# Poker Hand Checker

**System dependencies**

- Ruby 3.0.2
- Ruby on Rails 6.1.4.1
- Rspec 5.0.0
- rubocop-rails
- Yarn 1.22.17
- Bundler 2.2.31

Instructions to run locally

```
cd <PROJECT LOCATION>
rails server
```
Instructions to run Tests
```
Rspec spec
```
Access the website at: <http://localhost:3000>
